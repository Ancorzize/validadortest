Postman Runner API (Newman Validator)

Este proyecto es un servicio en Node.js + Express que permite ejecutar colecciones de Postman usando Newman y generar reportes automáticamente.

> Requisitos del servidor

Para ejecutar este proyecto en un servidor (VPS o máquina limpia), necesitas instalar:

1. Node.js
Versión mínima: 18
Recomendado: 20 LTS

Instalación:

curl -fsSL https://deb.nodesource.com/setup_20.x | sudo -E bash -
sudo apt install -y nodejs

Verificar:

node -v

npm -v"# validadortest" 
"# validadortest"
